# SublimeCustomizations
Arquivos de customização do Sublime Text.

## Atalhos
#### 1-) Salvar todos os documentos: ctrl+S
#### 2-) Salvar somente o documento que está sendo exibido: ctrl+shift+S
#### 3-) Escrever "console.log()" e mudar o foco para o meio dos parênteses: ctrl+shift+L
###### Obs.: Muita gente acha arriscado colocar o comando de salvar todos os arquivos diretamente no ctrl+S, então caso prefira, pode inverter os atalhos 1 e 2, invertendo o atributo "command" de seus jsons.

## Instruções para configuração

##### 1-) Acesse o menu Preferences > Key Bindings - User
##### 2-) Copie e cole o conteúdo do arquivo "Atalhos.json" dentro do arquivo aberto e salve.
